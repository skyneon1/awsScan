How to get AWS Credentials

  1. Log in to your AWS Management Console.
  2. In the top right corner, click your username and select Security credentials.
  3. Scroll down to the Access keys section and click Create access key.
  4. Select Command Line Interface (CLI) and confirm.
  5. Copy the Access key ID and Secret access key shown.
  6. (Alternatively) If you have the AWS CLI installed, you can upload your ~/.aws/credentials file.
